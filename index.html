HCJDemo & HTML 기본태그 실습

  • 실습 내용: Home, Timetable, Recipe, Product, Register, About, Contact 페이지 제작
  • 사용 기술: List(메뉴/레시피), Table(시간표), Multimedia(오디오/비디오), 입력 양식(사용자 등록)
  • 주요 기능: 배경색 변경 버튼 및 실시간 시간 출력 (JavaScript)

보안 수업 및 웹 프로그래밍 강의에서 학습한 다양한 HTML 태그들을 직접 적용해 본 프로젝트입니다. 표(Table)를 이용한 시간표 구성, 입력 폼(Form)을 활용한 등록 페이지, 멀티미디어 태그 삽입 등 웹 페이지의 필수 구성 요소들을 테마별로 나누어 구현했습니다.
